# Student Management System

A console-based Student Management System built using Python and SQLite database.

## Features
- Add student
- View all students
- Search student by roll number
- Delete student
- Persistent data storage using SQLite

## Technologies Used
- Python
- SQLite
- Git & GitHub
- VS Code

## How to Run the Project
1. Open terminal in project folder
2. Run the command:
   ```bash
   python student_db.py
